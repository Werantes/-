Система управления гостиницей

Описание проекта
Система для автоматизации процессов управления гостиницей:
- Бронирование номеров
- Регистрация гостей (check-in/check-out)
- Управление номерным фондом
- Финансовый учет и отчетность

Технологии
- Backend: Python/Java
- Database: PostgreSQL
- Frontend: HTML/CSS/JavaScript
